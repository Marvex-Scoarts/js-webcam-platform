# Jadexarts Cam: Web Camera Interface Platform

A sleek, modern client-side webcam capturing tool engineered with Vanilla JavaScript, semantic HTML5, and responsive CSS. 

## Technical Highlights
- **Asynchronous API Integration**: Utilizes the modern `navigator.mediaDevices.getUserMedia` browser API to request secure device camera streams.
- **Dynamic Matrix Rendering**: Captures live stream frames and maps them directly onto an HTML5 `<canvas>` element for processing.
- **Instant Client-Side Downloads**: Implements clean URL generation (`toDataURL`) allowing users to download snapshots securely without any backend storage database dependencies.

##  Installation & Deployment
1. Clone this repository to your local directory.
2. Open the `index.html` file in any modern mobile or desktop web browser.
3. Grant camera access permission when prompted to interact with the live media stream.
